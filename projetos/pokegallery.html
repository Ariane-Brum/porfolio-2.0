Projeto

PokeGallery.

Projeto desenvolvido para fins de estudo da biblioteca React JS com o consumo de uma API REST (PokeAPI).

  • Link do site
  • Link do repositório no Github

Tecnologias

  • ReactJs

    React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuários em páginas web.

  • Sass

    Sass (pré-processador css) é uma linguagem de folha de estilo. É uma simples linguagem de script usada em arquivos sass.

  • Axios

    Axios é um cliente HTTP baseado em promessa simples para o navegador e node.js.

  • React Icons

    O react-icons é uma biblioteca de icones famosos, tais como: Weather Icons, Material Design e Font Awesome.

Funcionalidades

  • Listagem dos pokémons
  • Botão que amplia a listagem dos pokémons
  • Método para filtrar os pokémons a partir do seu nome ou ID
  • NavTabs para as seções - [Stats] - [Biografy] - [Evolution]
  • Selecionar pokémon
